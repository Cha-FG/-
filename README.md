我设计了一个收纳盒，它的外形像一个松果（仅仅因为我喜欢它），因为我经常随便丢东西，之后找起来就特别的麻烦，所以我设计了这个收纳盒，每次我用完东西就随便丢在里面，然后当我需要的时候我只需要把松果的盖子一盖，它就能告诉我什么东西在哪里，需要一个感应元件，和一个声音提示器，然后当它感应到我需要的东西，它就可以告诉我这个东西在哪里，我就能很方便的找到它了
